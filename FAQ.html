What is meant by "nested" HTML elements

When you hear the term "nested" when we refer to HTML elements we are simply talking about elements that can be placed in other elemnts

What are the main two different types of HTML List elements

The two different types of list elements are ordered and unordered list. An ordered list displays the items in a order using letter of alphbets, while an unordered list only uses bullet points and does not have a specific order.

What is the purpose of the "href" HTML attribute for an anchor tag

The purpose of the "href" attribute is to specify the URL of the page the link goes to.

What is the purpose of the "scr" HTML attribute for an image tag

The purpose of the "src" attribute is to specify the file path of a resource, such as images or videos.

Which HTML attribute can be used to add cascading style sheets

Use the attribute style along with the link element to add CSS, short for cascading style sheets.
